**Дополнительные рекомендации**

**1. Двуязычность**

Для работы с иностранными производителями желательно сделать сайт минимум на русском и английском языках.

**2. SEO-структура**

Создайте отдельные посадочные страницы под запросы:

* «уполномоченное лицо ЕАЭС»;
* «authorized representative EAEU»;
* «сертификация ТР ТС»;
* «декларация соответствия ЕАС»;
* «представитель изготовителя в ЕАЭС».

**3. Доверие и комплаенс**

Разместите:

* реквизиты компании;
* членство в ассоциациях (если есть);
* сертификаты ISO;
* отзывы клиентов;
* информацию о конфиденциальности данных и хранении документации.

**4. Формы захвата заявок**

На каждой ключевой странице разместите короткую форму:

* тип продукции;
* страна изготовителя;
* код ТН ВЭД (если известен);
* контактные данные.
